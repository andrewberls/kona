An interface for defining time-based animations using sprite sheets. TODO: DOCS

Options:

  • sheet - (String) The path to the sprite sheet for the animation. Ex: 'img/entities/player_run.png'
  • width - (Integer) The width of each animation frame, in pixels. Defaults to the box width of the associated entity
  • height - (Integer) The height of each animation frame, in pixels. Defaults to the box height of the associated entity
